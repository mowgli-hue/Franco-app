# ChatGPT Board-Image Prompts — "French with Sophie" (F-02 → F-20)

How to use:
1. Paste the MASTER STYLE prompt + ONE lesson block into ChatGPT (image generation).
2. After it generates, ZOOM IN and check the French spelling + accents (é è à ç î). If any word is
   wrong, reply "fix the spelling: it should be '…'" and regenerate — image models sometimes garble
   French, so always proofread the board before using it.
3. For a consistent series look, tell ChatGPT "same teacher, same desk, same FRANCO style as before."
4. Use the finished image as the avatar's background "look" in HeyGen (Add a look).

Tip: keep boards SHORT (5–7 lines) — that's what kept your F-01 board's spelling clean.

### F-02
Badge: F-02 · Title: Les Sons du Français · Subtitle: The 8 Sounds
👂 E = "uh" (le)
🅰️ É = "ay" (café)
🗣️ R = from the throat (merci)
🤫 H = always silent (hôpital)
🌙 OU = "oo" (vous)
👄 U = round lips + "ee" (tu)
〰️ ON / AN = nasal (bon · grand)
